# FastAPI Feature Roadmap

## Phase 1: Core Social Features (Current Focus)

  - [ ] **Social Interactions Endpoints**
  - [ ] Like/Unlike a Post
  - [ ] Add a Comment to a Post
  - [ ] Delete a Post/Comment
- [ ] **Friendship System Endpoints**
  - [ ] Send Friend Request
  - [ ] Accept/Reject Friend Request
  - [ ] List Friends
  - [ ] Follow/Unfollow User
- [ ] **Profile Management**
  - [ ] Edit Profile (Bio, University, Profile Picture)
  - [ ] View Other User Profiles

## Phase 2: Advanced Feed & Discovery

  - [ ] **Smart Feed Algorithm**
  - [ ] Filter feed to show only Friends & Following
  - [ ] "Suggested Users" endpoint
  - [ ] **University Features**
  - [ ] Filter posts by University
  - [ ] Search users by University

## Phase 3: Search & Notifications

  - [ ] **Search**
  - [ ] Search Users by username
  - [ ] Search Posts by caption/content
  - [ ] **Notifications**
  - [ ] Notification model (Like, Comment, Friend Request)
  - [ ] Endpoint to get unread notifications

## Phase 4: Real-time Messaging

  - [ ] **Direct Messages**
  - [ ] WebSocket endpoint for real-time chat
  - [ ] Message history
